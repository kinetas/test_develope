6. dto/file/FileUploadResponse.java
   - 필드: fileId, originalName, fileType, fileSize, uploadedAt

7. service/FileStorageService.java
   - storeFile(MultipartFile, FileType, User): 확장자 검증, 크기 제한(50MB), UUID 저장명 생성, 파일 시스템 저장, DB 엔티티 반환
   - loadFileAsResource(String storedName): UrlResource 반환
   - deleteFile(Long fileId, User requester): 본인/ADMIN 권한 검증 후 파일 삭제

8. controller/FileController.java
   - 4개 엔드포인트 구현 (상세는 [API 엔드포인트] 섹션 참조)

9. resources/application.properties (수정)
   - file.upload-dir=uploads/
   - spring.servlet.multipart.max-file-size=50MB
   - spring.servlet.multipart.max-request-size=55MB 추가

[API 엔드포인트]
POST   /api/files/upload          파일 업로드 (multipart/form-data, params: file, fileType)
GET    /api/files/{fileId}        파일 다운로드 (attachment)
DELETE /api/files/{fileId}        파일 삭제 (본인 또는 ADMIN만 허용)
GET    /api/files/my              내가 업로드한 파일 목록 조회
