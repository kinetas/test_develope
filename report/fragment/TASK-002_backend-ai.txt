[API 엔드포인트]
POST /api/auth/register
  Request Body: { "username": "string", "email": "string", "password": "string" }
  Response 201: { "message": "User registered successfully", "username": "string", "id": number }
  Response 409: { "error": "Username/Email already taken" }

POST /api/auth/login
  Request Body: { "username": "string", "password": "string" }
  Response 200: { "token": "JWT_STRING", "username": "string", "role": "USER|ADMIN" }
  Response 401: { "error": "Invalid username or password" }

[JWT 구성]
- 알고리즘: HMAC-SHA (키 길이에 따라 HS256/HS384/HS512 자동 선택)
- Payload claims: sub(username), role, iat, exp
- 만료 시간: application.properties jwt.expiration-ms (기본 3600000ms = 1시간)
- 시크릿 키: application.properties jwt.secret (최소 32자 권장)
