## 기술 구현 세부사항
================================================================

WebSocket 프레임워크:
  - spring-boot-starter-websocket (pom.xml에 이미 선언됨)
  - STOMP 프로토콜, SockJS 폴백 (IE 등 구형 브라우저 지원)

메시지 흐름:
  클라이언트 → /app/chat.send
  → ChatController.handleChatMessage()
  → ChatService.saveMessage() (H2 DB 저장)
  → SimpMessagingTemplate.convertAndSend("/topic/session/{id}")
  → 모든 구독 클라이언트에 메시지 전달

주사위 로직:
  "d20" → parseDiceSides() → 20 → Random.nextInt(20)+1 → 1~20
  지원: d4, d6, d8, d10, d12, d20, d100 (임의 dN 모두 동작)

보안:
  - Principal 기반 sender 강제 (서버 측에서 덮어씀)
  - WebSocket 엔드포인트는 PUBLIC (JWT 없이 접속 가능)
  - 향후 개선: STOMP CONNECT 시 JWT 토큰 헤더 검증 추가 가능
